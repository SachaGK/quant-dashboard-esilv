# Quant Dashboard ESILV

**IF ESILV 2025-2026 • Python, Git, Linux for Finance**

Plateforme professionnelle d'analyse quantitative pour le trading et la gestion de portefeuille.

## Équipe

- **Sacha Guillou Keredan** - Quant A (Analyse d'actif unique)
- **Martin Partiot** - Quant B (Portefeuille multi-actifs)

## Objectifs

Développer une plateforme complète permettant :
- Analyse de stratégies de trading sur actifs uniques
- Construction et optimisation de portefeuilles diversifiés
- Calcul de métriques quantitatives professionnelles
- Visualisation des performances en temps réel

## Stack Technique

- **Backend** : Python, Flask, Pandas, yfinance
- **Frontend** : React, TypeScript, Recharts
- **Déploiement** : Linux VM avec cron jobs
